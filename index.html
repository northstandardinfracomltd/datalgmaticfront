Un processus simple, accessible à tous!

  • 1 IMDb : si ce n’est pas déjà fait, ajoutez votre titre sur IMDb (une filiale d’Amazon, c’est gratuit). Datalog est connecté à IMDb et récupère automatiquement vos informations. Si vous n’avez pas le temps, Datalog propose un service pour effectuer cette tâche à votre place.
  • 2 AWS : téléversez vos vidéos sur AWS (également une filiale d’Amazon, c’est gratuit). Datalog est connecté à AWS et récupère directement les liens de vos fichiers vidéo. Si le processus vous semble complexe, si vous manquez de temps ou si vos fichiers sont trop volumineux, Datalog vous offre la possibilité d’envoyer vos vidéos par voie postale.
  • 3 Datalog : complétez l’enregistrement en 5 étapes simples, pour moins de 5 minutes, grâce à notre formulaire d’ajout de titre.
  • 4 Suivez la publication de votre titre sur Amazon Prime Video, consultez vos statistiques d’audience et demandez des modifications si nécessaire via Datalog.

Manque de temps ? Peur du processus ? Difficultés techniques ? Des solutions existent.

Enregistrement IMDb

Enregistrer son titre sur IMDb permet à Amazon de récupérer toutes les informations essentielles (titre, synopsis, affiches, crédits, casting, genre, etc.). Le processus est simple mais peut être long. C’est pourquoi, lors de votre inscription sur Datalog, vous pouvez choisir notre service d’enregistrement sur IMDb : nous nous en occupons pour vous.

☁️

Téléversement sur AWS

Enregistrer son titre sur AWS consiste à déposer vos fichiers vidéo (bandes-annonces, épisodes, films, etc.) dans un espace cloud sécurisé. L’inscription et la configuration du bucket peuvent être complexes si vous n’êtes pas familier avec la console AWS, et le téléversement peut être long ou instable si vos fichiers sont volumineux ou si votre connexion Internet est limitée. C’est pourquoi Datalog vous propose un service complet pour transférer vos vidéos directement sur les serveurs AWS.
